def pip._vendor.lockfile.locked (   path,

Decorator which enables locks for decorated function.

Arguments:
 - path: path for lockfile.
 - timeout (optional): Timeout for acquiring lock.

 Usage:
     @locked('/var/run/myname', timeout=0)
     def myname(...):
         ...
